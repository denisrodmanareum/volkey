# OpenClaw 기반 자기개선형 15분 급등 단타 스캘핑 봇 (Testnet)

본 문서는 엔젤아름이 제시한 마스터 플랜을 운영 기준으로 고정하기 위한 요약 기준서다.

## 핵심 원칙
- Testnet 전용 운영
- 실시간 자동 자기수정 금지
- 운영 전략 / 실험 전략 분리
- 개선안은 검증 후 candidate로만 등록
- 손절/리스크 장치 제거 금지
- 텔레그램 리포트는 한글 고정

## 자기개선 루프
1. 실시간 거래 수행
2. 거래 로그/피처 스냅샷 저장
3. 일일 리뷰 에이전트 분석
4. 실패 패턴/개선 포인트 추출
5. 개선안 후보 생성
6. 자동 백테스트/워크포워드 검증
7. 통과 시 candidate 등록
8. 다음 세션 비교 실행

## 역할 분리
- 실시간 트레이더: 주문/포지션/리스크/로그
- 사후 개선 연구원(OpenClaw): 리뷰/패턴/개선/검증/레지스트리

## 전략 방향
- 15분 급등 탐지 + 1/3/5분 미세 타이밍
- 가짜 돌파 필터 강화
- 동적 손절/동적 익절/트레일링
- 포지션 정보 상세 기록 (진입/청산/MFE/MAE/근거)

## 고정 분석 스케줄 (KST)
- 13:30 중간 체결 분석 + 당일 보완 패치 제안
- 21:30 종합 분석 + 다음날 전략 패치안

## 상태정보창(10분)
- 시스템 상태, 시장 상태, 전략 상태, 포지션 상태, 리스크 상태, 성장 상태를 포함
- 누적수익/일일손익/MDD/연속손실을 항상 포함

## 실행 메모
- 본 문서는 요약판이며, 상세 설계는 엔젤아름이 전달한 원문을 우선 기준으로 한다.
